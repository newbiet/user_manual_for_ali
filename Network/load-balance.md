# 14.4.6 负载均衡

系统在云路由模式下，支持负载均衡功能。
负载均衡，即将负载（工作任务，访问请求）进行平衡、分摊到多个操作单元（服务器，组件）上进行执行。在系统云路由上，其逻辑模型如下：
